# Create Balloon (机械动力：气球)

为机械动力航空学添加充气热气球。一个方块一个气囊，空格即飞。

## 构建

1. 将前置模组 JAR 放入 `mod flies/`：
   - Create 1.21.1-6.0.10+
   - Create: Aeronautics Bundled 1.21.1-1.1.3+
   - Sable 1.21.1-1.1.3+

2. 放入 `libs/`：
   - sable-companion-common-1.21.1-1.5.0.jar

3. 运行：
```bash
./gradlew build
```

JAR 输出在 `build/libs/`。
